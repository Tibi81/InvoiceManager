# Invoice Manager - Overview

## Project Goal
Personal invoice management system that can:
- process invoice emails from Gmail
- manage manual recurring invoices
- provide invoice tracking and payment helpers

## Current Scope
- Backend: Flask REST API (Python 3.11+)
- Web frontend: React + Vite
- Desktop frontend: Flet (Python)
- Database: SQLite

## MVP Focus
1. Email-based invoice ingestion and parsing
2. Manual recurring templates
3. Invoice list with unpaid/paid/all views
4. Mark paid workflow and optional payment helpers (QR/payment link)
